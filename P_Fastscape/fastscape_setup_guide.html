Fastscape Practical Fastscape Landscape Evolution – Student Setup Guide

Follow these steps to install a reliable, class-wide environment for the Fastscape landscape evolution practical. Commands are provided for Windows, macOS, and Linux.

What you’ll need
  • The course repository from GitHub: LandscapeDynamics Repository
  • This environment file: initiate.yml (located in the P_Fastscape folder of the repository)

Download the Course Repository

Option A: ZIP download (simplest)

  1. Go to LandscapeDynamics GitHub repository.
  2. Click Code ▸ Download ZIP.
  3. Unzip to a folder you can find, e.g., Documents\\fastscape-practical.

Option B: Clone with Git

git clone https://github.com/Richard-Ott/LandscapeDymamics.git
cd LandscapeDymamics

Step 1 — Install Miniforge or Miniconda (one-time)

Install Miniforge for your OS (Windows/macOS/Linux). It’s a minimal Conda distribution preconfigured for the conda-forge channel.

  • Miniforge: conda-forge.org/download
  • Miniconda alternative: Miniconda installer

Already have Anaconda/Miniconda? You can keep it. Using Miniforge is recommended for reproducible, conda-forge-only environments.

(Nice to have) Install mamba in base

Run this command in Anaconda Prompt (Windows) or Terminal (macOS/Linux):

conda install -n base -c conda-forge mamba

Step 2 — Locate the Environment File

The initiate.yml file is provided in the P_Fastscape folder of the repository you downloaded. Use this file to create your environment.

Step 3 — Create the Conda Environment

Windows

  1. Open Anaconda Prompt.
  2. Navigate to the P_Fastscape folder inside the repository:
    3. cd %HOMEPATH%\Documents\LandscapeDymamics\P_Fastscape
  3. Create the environment (name it fastscape-class): 
    mamba env create -f initiate.yml -n fastscape-class

    If you don’t have mamba yet, use:

    conda env create -f initiate.yml -n fastscape-class
  4. Activate it and launch Jupyter: 
    conda activate fastscape-class
jupyter notebook

macOS / Linux

  1. Open the Terminal app.
  2. Go to the P_Fastscape folder inside the repository: 
    cd ~/Documents/LandscapeDymamics/P_Fastscape
  3. Create the environment (name fastscape-class): 
    mamba env create -f initiate.yml -n fastscape-class

    No mamba?

    conda env create -f initiate.yml -n fastscape-class
  4. Activate and launch Jupyter: 
    conda activate fastscape-class
jupyter notebook
Tip: You can also start JupyterLab if included: jupyter lab.

Step 4 — Open the Practical Notebook

  1. In the Jupyter file browser, navigate into the folder you downloaded (e.g., LandscapeDymamics/notebooks).
  2. Open the practical notebook (e.g., 01_fastscape_practical.ipynb).
  3. The import check will run automatically in the first lines of the notebook when you import all the modules.

Prepared for the Fastscape practical. If you hit an issue not listed here, share the exact error message and your OS in the class forum/issue tracker.
